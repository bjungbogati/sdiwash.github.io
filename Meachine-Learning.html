Definition:¶

Principal component analysis (PCA) is a statistical procedure that uses an orthogonal transformation to convert a set of observations of possibly correlated variables into a set of values of linearly uncorrelated variables called principal components (or sometimes, principal modes of variation).#Wikipedia

How It works:¶

PCA is one of the most popular techniques for dimensionality reduction. It models a linear subspace of our data by capturing its greatest variability. Stated differently, it accesses our dataset's covariance structure directly using matrix calculations and eigenvectors to compute the best unique features that describe our samples.

An iterative approach to this would first find the center of our data, based off its numeric features. Next, it would search for the direction that has the most variance or widest spread of values. That direction is the principal component vector, so it is then added to a list. By searching for more directions of maximal variance that are orthogonal to all previously computed vectors, more principal component can then be added to the list. This set of vectors form a new feature space that we can represent our samples with.

In the above 2D dataset, the orange line represents the principal component direction, and the gray line represents the second principal component. The one that's going to get dropped:

1.By dropping the gray component above, the goal is to project the 2D points onto 1D space. Move the original 2D samples to their closest spot on the line.

2.Once you've projected all samples to their closest spot on the major principal component, a shadow, or lower dimensional representation has been formed.

3.The summed distances traveled by all moved samples is equal to the total information lost by the projection. An an ideal situation, this lost information should be dominated by highly redundant features and random noise.

def do_PCA(armadillo, svd_solver):
    from sklearn.decomposition import PCA
    pca = PCA(n_components = 2) 
    pca.fit(armadillo)
    result = pca.transform(armadillo)
    return result

def do_RANDOMPCA(armadillo):
    from sklearn.decomposition import RandomizedPCA
    rpca = RandomizedPCA(n_components = 2)
    rpca.fit(armadillo)
    result1 = rpca.transform(armadillo)
    return result1
